# PUSH_SWAP

Push Swap is an algorithm challenge where the objective is to sort a stack of integers using the least number of operations possible, restricted to an auxiliary stack and a limited set of allowed movements.

## Compilation:
	make
	make clean

## Usage:
	./push_swap (numbers)

# CHECKER

A checker to make your life easier to verify the push_swap operations.

## Compilation:
	make bonus
	make clean

## Usage:
	./push_swap (numbers) | ./checker (numbers)
	or
	./checker (numbers) [enter]
	write operations ... Ctrl + D when you finish.
